Proof-of-Diversity (PoDiv)

🌐 What is PoDiv?

A revolutionary consensus mechanism that validates transactions based on network diversity rather than computational power or stake.

⚡ How It Works:

  • • Validators are selected based on geographic, device, and network diversity
  • • Higher diversity score = higher validation probability
  • • Prevents centralization and promotes global participation
  • • Rewards come from PiCoin integration and network fees
